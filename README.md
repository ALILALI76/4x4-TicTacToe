# 4x4-TicTacToe
4x4 TicTacToe game programmed in Java only

PLAY TICTACTOE ON A 4X4 SQUARE WITH 2 PLAYERS!

**How the game works:**

You are greeted with a welcome message and a short explanation of the game.

Symbols 'X' and 'O' are alternately placed in squares 1-16 until there are 3 of the same symbols next to each other 
either vertically, horizontally or diagonally in which case the symbol looses or until all 16 squares are occupied which results in a draw.

Symbols are placed by typing in the number of the desired square into the console and confirmed by pressing the enter button.

You play with 2 players and you may alternate who starts the game as 'X'.

**HAVE FUN PLAYING!**
